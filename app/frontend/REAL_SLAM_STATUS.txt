🔧 HOW TO CONFIGURE

1. Set Camera URLs in backend_interface.py config:

   config = {
       "left_camera_url": "http://192.168.1.47:8080/video_feed",
       "right_camera_url": "http://192.168.1.48:8080/video_feed"
   }

2. Or pass directly to RealSLAMViewer:

   viewer = RealSLAMViewer(
       parent=root,
       left_camera_url="http://camera1/stream",
       right_camera_url="http://camera2/stream"
   )

3. Supported formats:
   - MJPEG IP camera: http://192.168.1.47:8080/video_feed
   - USB camera: "0" or "1"
   - Video file: "video.mp4"
   - RTSP stream: "rtsp://..."

📈 REAL PROCESSING PIPELINE

For each frame pair:

1. Feature Extraction (SIFT)
   - Extract ~2000 features per frame
   - Compute 128-dimensional descriptors
   - Score feature quality

2. Stereo Matching
   - Match left-right features using FLANN
   - Apply Lowe's ratio test
   - Filter outliers with RANSAC

3. 3D Reconstruction
   - Compute Essential matrix (5-point algorithm)
   - Perform triangulation
   - Generate 3D point coordinates

4. Pose Estimation
   - Use PnP solver with RANSAC
   - Estimate SE(3) transformation
   - Update camera trajectory

5. SLAM System
   - Create keyframes when motion threshold exceeded
   - Accumulate point cloud
   - Detect loop closures
   - (Optional) Bundle adjustment

6. Visualization
   - Render camera feeds
   - Draw 2D occupancy map
   - Project 3D point cloud
   - Display real statistics
